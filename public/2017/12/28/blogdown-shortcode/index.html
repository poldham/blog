Add Shortcodes to a Hugo blog in blogdown

2017/12/28

I have been planning to transfer my existing Jekyll based blog over to blogdown and Hugo for quite a while. Having spent most of last year hopping on and off aeroplanes this had turned into an epic fail. Christmas seemed like a good time to fix that.

The blogdown package from Yihui Xie generates a Go based Hugo blog. Once you have installed blogdown and followed the nice and friendly instructions in the blogdown book you should be ready to go.

I have used the example theme that has been edited by Yihui as I didn’t want to get stalled on choosing and editing a theme. But, I do commonly want to add things such as links to twitter, images or videos or dashboards into my blogs.

Some of these things can easily be done in code chunks inside Rmarkdown docs, but others are more fiddly. This is where the Hugo shortcodes come in very handy.

Here, I’ll just cover a few of the shortcodes that I think many people might want to use.

The blogdown book has a section on shortcodes and for a full list of shortcodes see the Hugo Shortcodes site. If you want something more hands on then Giraffe Academy has some nice Hugo tutorials.

The blogdown shortcode function

The blogdown::shortcode() function addresses potential problems with pandoc conversion in R Markdown. The problem appears to be that Pandoc will convert special characters that Hugo will then not be able to recognise. I am not so sure how common this problem is as I found that some shortcodes render just fine. It may be that this is a dynamic situation as Hugo now recognises Pandoc.

Add a video

I recently gave a presentation to governments that was heralded in advance by the host as “stunning”. I was nonplussed by this and spent lunchtime pondering how not to let the audience down with the rather pedestrian talk I would be giving.

This reminded me of Tenacious D and Tribute. So I began my presentation with the words “This is not the most stunning presentation in the world, its just a tribute”.

If I had wanted to do this in a blog post I could have simply entered the following shortcode into the Rmarkdown or markdown in RStudio, where _lK4cX5xGiQ is the id for the You Tube Tribute video.

{{<youtube _lK4cX5xGiQ>}}

A safer version of this would be to use a code chunk, but it doesn’t seem to work.

blogdown::shortcode("youtube", src = "_lK4cX5xGiQ")

Note that there should be no space between the opening and closing angle bracets or this won’t work.
